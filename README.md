1. Generator          ← учишься создавать канал-источник данных
2. Pipeline           ← соединяешь генераторы в цепочку стадий
3. Fan-out            ← распределяешь одну стадию на N воркеров
4. Fan-in             ← собираешь результаты обратно в один канал
5. Worker Pool        ← fan-out с фиксированным пулом + очередь задач
6. Done channel       ← учишься останавливать всё вышеперечисленное
7. Tee channel        ← разветвляешь поток когда нужно два потребителя
