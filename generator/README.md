**Генератор** — это функция, которая не возвращает все значения сразу, а производит их по одному, по требованию (lazily), приостанавливая своё выполнение между выдачами.

```
создаёт канал  
возвращает его сразу синхронно
и в фоне запускает асинхронные вычисления (в горутине)
```


В Python это выглядит так:
```python
def generator():
    yield 1
    yield 2
    yield 3
```
В Go нет yield и приостановки выполнения функции как языковой конструкции. Но горутина + канал дают функционально то же самое поведение:
```go
func generator() <-chan int {
    out := make(chan int)
    go func() {
        for i := 1; i <= 3; i++ {
            out <- i // это как "yield i" — горутина блокируется здесь,
                      // пока кто-то не прочитает значение
        }
    }()
    return out
}
```
